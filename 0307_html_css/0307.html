新尖兵上課準備步驟

  1. 按下電腦主機電源鍵
  2. 選取「考試分類」
  3. 填寫密碼：1234
  4. 打開 Webex
  5. 登入 Webex
  6. 進入 Webex 會議室

注意事項

  • 上課有問題請隨時舉手詢問
  • 教室內請勿飲食
  • 多跟老師眼神交流，讓老師知道你的狀況
  • 請假請事先跟班導報備
  • 若有網路延遲嚴重問題，可以即時舉手版應
